# 🌐 Python Port Scanner

A simple Python-based port scanner to identify open ports on a target system.

## 🚀 Features

* Scans common ports
* Detects open and closed ports
* Lightweight and fast
* Beginner-friendly cybersecurity project

## 🛠️ Tech Used

* Python
* Socket Programming

## ▶️ How to Run

```bash
python scanner.py
```

## 📌 Example

```
Enter target IP or domain: scanme.nmap.org

Scanning target: scanme.nmap.org
=================================
[OPEN] Port 22
[CLOSED] Port 80
[OPEN] Port 443
```

To understand basic networking and port scanning concepts used in cybersecurity.
